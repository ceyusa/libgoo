GooList

typedef struct _GooList GooList;

An iterable list type

GooIterator

typedef struct _GooIterator GooIterator;

An iterator for a GooList is used for iterating over a list. The iterator is an instance that keeps the position state information. The GooList instance will not by itself keep a position state. Only iterators can keep a position state for a list.

goo_iterator_get_current ()

GooObject*          goo_iterator_get_current            (GooIterator *self);

Does not move the iterator. Returns the objecto at the curren position. If there's no current position, this method returns NULL. If not NULL, the returned value must be unreferenced after use.

self : An GooIterator instance
Returns : the current object or NULL

goo_iterator_is_done ()

gboolean            goo_iterator_is_done                (GooIterator *self);

Does the iterator point to some valid list item? You can use this property to make loops like:

Example: 

GooList *list = goo_simple_list_new ();
GooIterator *iter = goo_list_create_iterator (list);
while (!goo_iterator_is_done (iter))
{
   GooObject *cur = goo_iterator_get_current (iter);
   ...
   g_object_unref (G_OBJECT (cur));
   goo_iterator_next (iter);
}
g_object_unref (G_OBJECT (iter));
g_object_unref (G_OBJECT (list));

self : An GooIterator instance
Returns : TRUE if it points to a valid list item, FALSE otherwise

goo_iterator_get_list ()

GooList*            goo_iterator_get_list               (GooIterator *self);

Does not move the iterator. Returns the list of which this iterator is an iterator. The returned list object should be unreferenced after use. Remember when using this property that lists shouldn't change while iterating them.

self : An GooIterator instance
Returns : The GooList instance being iterated
