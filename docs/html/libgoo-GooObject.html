GooObject

typedef struct {
	GObject parent;
	
	gchar* name;
	gchar* prefix;
	GooObject* owner;
} GooObject;

The GooObject structure. Use the functions to update the variables.

GObject parent;
gchar *name; the object's name. It is used for logging purposes.
gchar *prefix;

goo_object_get_name ()

gchar*              goo_object_get_name                 (GooObject *self);

Returns a copy of the name of object. Caller should g_free() the return \ value after usage. For a nameless object, this returns NULL, which you can safely g_free() as well.

self : An GooObject instance
Returns : the name of object. g_free() after usage. MT safe.

goo_object_set_name ()

void                goo_object_set_name                 (GooObject *self,
                                                         const gchar *name);

Set the name of object. This function makes a copy of the provided name, so the caller retains ownership of the name it sent.

self : An GooObject instance
name : An gchar with the object's name

goo_object_get_owner ()

GooObject*          goo_object_get_owner                (GooObject *self);

Gets the object's owner. This functions increases the refcount of the owner object so you should unref it after usage.

self : An GooObject instance
Returns : The object's owner, this can be NULL if object has no parent. unref owner after usage. MT safe.

goo_object_set_owner ()

void                goo_object_set_owner                (GooObject *self,
                                                         GooObject *owner);

Sets the owner of the object. The owner reference count will be incremented.

self : An GooObject instance
owner : The new GooObject owner object
