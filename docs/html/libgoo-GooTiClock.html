GooTiClock

GooTiClock — This class represents an TI OMX clock.

Synopsis




                    GooTiClock;

Description

This class represents an TI OMX clock. It is used by some other TI OMX components to synchronize each other.

The instance of this class must be unique in all the TI OMX components, So it is managed by the GooComponentFactory. The client code should not modify or access to this object.
